# Start hier (Schnellstart)

Wenn Sie nur **ein Habitat erfassen** möchten, reicht dieses Kapitel.
Alles Weitere können Sie später nachlesen.

## Was Sie brauchen

- **Smartphone** (empfohlen) oder PC/Laptop
- **Internetverbindung**
- **1–4 Fotos** vom Habitat (siehe Kapitel „Bilder erfassen“)
- Optional: Zugang zu einer **Organisation** (falls Sie einer zugeordnet werden)

## Dauer (als grobe Orientierung)

- **Standort zeichnen**: 1–3 Minuten
- **Fotos hochladen**: 2–5 Minuten
- **Analyse abwarten**: 10–60 Sekunden (je nach Verbindung)

## Ablauf in 5 Schritten

1. **Willkommen & Hinweise** (kurz lesen)
2. **Standort bestimmen** (Polygon auf der Karte zeichnen)
3. **Bilder erfassen** (Panorama + Detail + 1–2 Pflanzen)
4. **Habitat analysieren** (Ergebnis prüfen, ggf. Hinweis schreiben)
5. **Verifizieren** (nur Expert*innen) / sonst Abschluss

## Tipp vor dem Start

- Machen Sie das **Panoramafoto** am besten zuerst (mit sichtbarem Horizont).
- Falls am Handy: Standortfreigabe im Browser erlauben, wenn gefragt.
