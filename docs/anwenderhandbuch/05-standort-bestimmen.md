# Standort (Phase) – Schritte „Standort finden“, „Umriss zeichnen“, „Standortdaten ermitteln“

Hier legen Sie die Fläche fest, auf die sich Ihre Erfassung bezieht.
Sie zeichnen dazu ein **Polygon** (eine umrandete Fläche) direkt auf der Karte.

![Standort bestimmen (Screenshot folgt)](images/05-standort.png)

Dieses Kapitel deckt im Assistenten diese Schritte ab:

- **Standort finden**
- **Umriss zeichnen**
- **Standortdaten ermitteln**

## So gehen Sie vor (Schritt für Schritt)

### 1) Standort finden

Ziel: Sie bringen die Karte auf die richtige Stelle.

1. Zoomen Sie zur Fläche, die Sie erfassen möchten.
   - Am Smartphone: mit zwei Fingern (Auseinanderziehen/Zusammenziehen).
   - Am Desktop: mit **Mausrad** oder den **+ / –** Buttons.
2. Verschieben Sie die Karte, bis der Standort gut sichtbar ist.
3. Wenn der Ausschnitt passt: klicken Sie unten auf **„Weiter“**.

### 2) Umriss zeichnen

Ziel: Sie zeichnen die genaue Fläche als Polygon.

1. Setzen Sie Punkt für Punkt entlang der Fläche.
2. Schließen Sie den Umriss, indem Sie den ersten Punkt erneut antippen/anklicken.
3. Wenn der Umriss fertig ist: klicken Sie unten auf **„Weiter“**.

### 3) Standortdaten ermitteln (kurz prüfen)

Jetzt werden Standortdaten automatisch berechnet (z.B. Gemeinde, Flurname, Höhe, Hang, Exposition, Kataster).

1. Warten Sie kurz, bis die Werte angezeigt werden.
2. Prüfen Sie, ob die Angaben plausibel sind.
3. Wenn alles passt: klicken Sie unten auf **„Weiter“**.

## Achtung (häufige Stolpersteine)

- **Bereits erfasste Flächen**: Wenn Sie schon Polygone sehen, erfassen Sie die gleiche Fläche nicht noch einmal.
- **Falsches Polygon**: Wenn der Umriss falsch ist, beginnen Sie neu (z.B. „Neu beginnen“ / Umriss neu zeichnen).
- **Standortfreigabe**: Am Smartphone ist die Karte oft genauer, wenn der Browser Zugriff auf den Standort hat.

## Tipp

### Standortfreigabe am Smartphone (wenn die Karte ungenau ist)

Wenn NatureScout nach dem Standort fragt, wählen Sie **„Zulassen“**.
Je nach Handy/Browser sieht das etwas unterschiedlich aus.

Wenn Sie aus Versehen **„Blockieren“** gewählt haben:

- Öffnen Sie im Browser das Menü (meist „⋮“ oder „aA“).
- Gehen Sie zu **Website‑Einstellungen** / **Berechtigungen**.
- Setzen Sie **Standort** auf **Zulassen**.
- Laden Sie die Seite neu.

### Genauigkeit verbessern (optional)

- Wenn möglich: **„Genauer Standort“ / „Präziser Standort“** erlauben.
- Draußen ist GPS oft genauer als in Gebäuden.
- Wenn es weiterhin ungenau ist: Karte manuell verschieben und stark hineinzoomen.

Wenn Werte (z.B. Höhenlage) offensichtlich falsch sind:

1. Karte nochmal prüfen (richtige Fläche? richtig gezoomt?).
2. Umriss neu zeichnen (z.B. „Neu beginnen“).
3. Danach wieder mit **„Weiter“** fortfahren.
